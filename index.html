Why AvyOS?

🦫

Pure Go

Built with CGO_ENABLED=0. No C toolchain required. The entire userspace compiles with just the Go compiler.

🔒

Immutable Core

System files in /avyos/ are read-only. Updates are atomic. Configuration overrides live separately.

📦

Non-POSIX

Clean, modern interfaces without decades of legacy baggage. Linux compatibility via containers when needed.

🖥️

TUI Desktop

Terminal-based tiling window manager with BSP, grid, and spiral layouts. Like tmux meets i3.

🛡️

Capability-Based

Applications run with capability restrictions. Fine-grained permissions without root/non-root dichotomy.

⚡

Fast Development

Go's simplicity makes building and extending the OS enjoyable. Single-developer friendly codebase.

Filesystem Hierarchy

/
├── avyos/ # Immutable system (squashfs)
│ ├── cmd/ # Commands (nosuid, nodev)
│ ├── config/ # Default configs (noexec)
│ ├── data/ # Icons, fonts (noexec)
│ ├── apps/ # User applications
│ └── services/ # System services
├── config/ # Mutable config overrides
├── users/ # Home directories
├── cache/ # Runtime & kernel filesystems
│ ├── kernel/ # procfs, sysfs, devtmpfs
│ └── runtime/ # Runtime state (/run)
└── linux/ # Linux compatibility containers
